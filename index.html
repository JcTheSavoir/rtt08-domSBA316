Thinking about what site to build for an SBA project?

Look no further, this page will give you a start.

First Step, write down 5 basic ideas

  • Build a quiz type game. (personality test, you think you know [blank] show?, etc)
  • A sales page (local buisness, your own business, or madeup one)
  • Page that links to past projects you've worked on. You can include navigation and brief description for each one.
  • Organizational helper (listing chores to do, scheduling due dates, to-buy list, etc)
  • A "how to" page of starting a programming journey. Could have a dropdown list for picking languages or type of jobs, with explanations on how to start and links to resources

Now that you have five ideas down, choose four and then three and then two, and finally pick the one you want. Don't take longer than a minute for each step of this. No second guessing, as there is no wrong answer

Time to put that idea into action (in a small way)

Only create one instance of the idea to start, otherwise you may overwhelm yourself. As in you would build one question for your quiz, the top section of your sales page, or a single project and description for a project index page. For this demonstration, we will be doing either the quiz or Organizational Helper example
